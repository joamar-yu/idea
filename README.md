# idea — 實踐你的idea | 需求釐清 & Prompt 生成

> 在動手之前，先把事情想清楚。

一個 Claude Code skill，透過結構化對話釐清你的真實需求，最終產出一份**需求說明書** + 一份可直接使用的**高品質 Prompt**。

---

## 解決什麼問題

你有個想法，直接丟給 AI 執行，結果跑偏了——不是 AI 不夠聰明，而是需求本來就沒說清楚。

`/idea` 強迫你在執行前先想清楚：目標是什麼、什麼算失敗、最難的地方在哪裡。

---

## 使用方式

在 Claude Code 中輸入：

```
/idea
```

skill 會引導你完成三個階段：

| 階段 | 內容 |
|------|------|
| **Step 1** | 透過對話，從 10 個面向拆解任務 |
| **Step 1.5** | 評估是否需要思考鏈、階段驗證、信心標記等推理策略 |
| **Step 2** | 產出結構化需求說明書，確認後再繼續 |
| **Step 3** | 輸出七大組件的高品質 Prompt，支援迭代修改至定版 |

---

## 安裝

將 `idea.md` 放入 Claude Code 的 skills 目錄：

```
~/.claude/skills/idea.md
```

---

## 適用情境

- 有模糊想法但不確定怎麼下 prompt
- 任務複雜、有多個子步驟或依賴關係
- 過去給 AI 的指令常常跑偏

---

## 語言支援

需求說明書與 Prompt 預設跟隨使用者語言輸出。使用英文對話即輸出英文；中文對話輸出中文，並附上英文版轉換提示。
